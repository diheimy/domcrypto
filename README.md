# dom-crypto

> Bot fullstack de arbitragem de criptomoedas (Spot/Futuros)

## 🚀 Quick Start

```bash
git clone <url-do-repositorio>
cd dom-crypto
cp .env.example .env
# Edite .env com suas variáveis reais
npm install
npm run dev
```
